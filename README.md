# 🗺️ Guia Turístico Web: O Melhor da Cidade do Recife

Projeto prático desenvolvido como parte do aprendizado de desenvolvimento front-end (HTML5 e CSS3). O objetivo do projeto foi construir uma página informativa e atrativa, simulando uma resenha cultural e turística para visitantes que desejam conhecer a capital de Pernambuco.

---

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica de dados, tabelas de informações, listas e hiperlinks externos.
* **CSS3:** Estilização visual, layout responsivo e aplicação da identidade visual inspirada nas cores da bandeira de Pernambuco.

---

## 📌 Requisitos Implementados no Projeto

O projeto foi construído seguindo critérios rigorosos de organização de conteúdo:

1. **Dados Demográficos:** Apresentação do nome oficial e população baseada em dados reais do Censo do IBGE.
2. **Gastronomia Regional:** Destaque para pratos típicos locais como o Bolo de Rolo, Cartola e Arrumadinho.
3. **Pontos Turísticos e Culturais:** Mapeamento visual das principais atrações, incluindo Recife Antigo e a Praia de Boa Viagem.
4. **Calendário Cultural:** Apresentação de comemorações específicas como o Carnaval (Galo da Madrugada) e o São João.
5. **Curadorias e Indicações (Links Úteis):** Links diretos para os melhores estabelecimentos gastronômicos e passeios da cidade.
6. **Mídia:** Inclusão de no mínimo 3 fotos ilustrativas integradas ao layout.

## 💻 Como Executar o Projeto Localmente

Como o projeto ainda não está hospedado em um servidor web, você pode baixá-lo e rodá-lo direto no seu computador seguindo estes passos simples:

### Opção 1: Baixando o arquivo ZIP (Mais fácil)
1. No topo desta página do GitHub, clique no botão verde **Code**.
2. Clique em **Download ZIP**.
3. Extraia o arquivo `.zip` em uma pasta no seu computador.
4. Abra a pasta extraída e dê um **duplo clique no arquivo `index.html`**. O projeto abrirá imediatamente no seu navegador de internet (Chrome, Edge, etc.).
